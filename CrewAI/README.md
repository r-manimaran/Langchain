## CREW AI
- Perform Multi-agent Orchestration
- Combines multiple autonomous agents into a "team"
- Agent: LLM based system that act autonomously by prompting itself (like a recursive function)
- Agents uses the Tools to complete Tasks like Webbrowsing
- With CrewAI we can created multiple agents that can perform complex sequential tasks
- CrewAI is build on top of LangChain, so the syntax are easy to use
- CrewAI works with OpenAI and other OpenSource models
